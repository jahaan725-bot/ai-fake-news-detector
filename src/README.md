# 📰 AI Fake News Detector

## Overview

AI-powered application that analyzes news articles and predicts whether they are likely real or fake.

## Features

- Fake News Detection
- Credibility Scoring
- Explainable AI Output
- Streamlit Dashboard

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- NLP

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
